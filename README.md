Learning Python through Python for Everybody (Dr. Chuck).
Tracking progress from zero to functional programs.

## Programs Written

### hello.py
First ever Python program — prints Hello World.
- **Concepts used:** print()
- **Written:** April 2026

### floor_converter.py
Converts a floor number between European and French numbering systems.
- **Concepts used:** input(), int(), variables, print()
- **Written:** April 2026

### pay_calculator.py
Calculates total pay with 1.5x overtime for hours over 40.
- **Concepts used:** input(), int(), float(), if/else, print()
- **Written:** April 2026

### pay_calcuv2.py
Upgraded pay calculator using a function. Applies 1.5x overtime for hours over 50.
- **Concepts used:** def, functions, return, if/else
- **Written:** April 2026

### number_check.py
Checks whether a number is positive, zero, or negative.
- **Concepts used:** input(), int(), if/elif/else
- **Written:** April 2026

### num_check_v2.py
Improved number checker with try/except error handling for invalid input.
- **Concepts used:** try/except, quit(), if/elif/else
- **Written:** April 2026

### grade_marker.py
Takes a grade between 0.0–1.0 and prints the corresponding letter grade (A–F).
- **Concepts used:** try/except, float(), if/elif/else
- **Written:** April 2026

### largestvalue.py
Accepts numbers in a loop until "done" — prints the largest value entered.
- **Concepts used:** while loop, break, float(), None, conditionals
- **Written:** April 2026

### smallestvalue.py
Accepts numbers in a loop until "done" — prints the smallest value and total count.
- **Concepts used:** while loop, break, float(), None, conditionals, count
- **Written:** April 2026

### loop1.py
Loops asking for numbers, breaks when a negative number is entered.
- **Concepts used:** while True, break, int(), conditionals
- **Written:** April 2026

### exploringstrings.py
Explores Python string methods using a sample string.
- **Concepts used:** count(), find(), replace(), strip(), startswith(), slicing, \n
- **Written:** April 2026

### file_handling.py
Opens a text file and counts the number of lines in it.
- **Concepts used:** open(), for loop, count, encoding
- **Written:** April 2026
